# IBM® Security Trusteer Rapport™
IBM® Security Trusteer Rapport™ is a free software that focuses on multi-layered protection against financial malware and complements your existing anti-virus software. It gives you extra protection when using these ANZ digital channels.
No anti-malware software will completely guarantee against unauthorised access or virus contamination but may increase the security of your computer systems. Anti-malware software should be used in addition to anti-virus software and in conjunction with your organisation’s IT program for reviewing your computer systems for malware. IBM Trusteer Rapport should be updated in accordance with any notifications from IBM.
EndNoIndex ## How to install
It is easy to install and will work transparently in the background without affecting your computer’s performance.
For more information, to download the software or for technical support, visit Trusteer Rapport and select ANZ BANK from the drop down list.
Please ensure you carefully read the IBM End User Licence Agreement.
## Accessing IBM support
For assistance with Trusteer Rapport, visit the Trusteer Rapport Technical Support site. Here you can access FAQs, manuals, tutorials, videos and connect with IBM experts 24/7.
